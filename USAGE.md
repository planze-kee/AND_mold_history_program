# 금형이력카드 처리 프로그램 - 사용 설명서

## 개요

HWP 형식의 금형이력카드에서 데이터와 이미지를 자동으로 추출하여 엑셀 DB로 관리하고, 템플릿 워드 문서에 자동 입력하는 통합 솔루션입니다.

---

## 시작하기

```bash
pip install -r requirements.txt
python main.py
```

### 시스템 요구사항
- Python 3.7+, Windows
- Microsoft Word 설치 (PDF 변환 기능 사용 시)

---

## 탭 구성

| 탭 | 기능 |
|----|------|
| 문서 생성/동기화 | XLSX → Word 일괄 생성, XLSX 변경 동기화 |
| 이력 관리 | 유지보수 이력 기록·조회·Word 반영 |
| PDF 변환/병합 | DOCX → PDF 변환 및 병합 |
| HWP 처리 | HWP → XLSX DB 변환 |
| 이미지 추출 | HWP → 이미지 추출 |

---

## 탭별 사용 방법

### 탭 1 — 문서 생성/동기화

**DOCX 일괄 생성**

1. DB 엑셀 파일 선택 (`00.DB_19-000.xlsx`)
2. 템플릿 파일 선택 (`data/templates/Word_양식.docx`)
3. 이미지 폴더 지정 (추출된 이미지가 있는 폴더)
4. 출력 폴더 지정
5. 처리 행 제한: `0` = 전체, `5` = 처음 5행만 (테스트용)
6. "DOCX 생성" 클릭

**동기화**

XLSX DB를 수정한 후 기존 Word 파일에 변경사항을 반영합니다.

1. 동일한 DB / 템플릿 / 이미지 / 출력 폴더 설정
2. "동기화" 클릭

---

### 탭 2 — 이력 관리

유지보수 내역을 기록하고 Word 파일에 반영합니다.

1. **설정**: DB 엑셀, 템플릿, 이미지 폴더, 출력 폴더 지정
2. **파일 목록**에서 대상 Word 파일 선택
3. 이력 내용 입력 (텍스트 에디터)
4. **"저장"**: `.data/<연번>_history.txt` + XLSX `事 由` 컬럼 갱신
5. **"Word에 반영"**: 이력 포함하여 Word 파일 재생성 (이미지도 다시 탐색·삽입)

---

### 탭 3 — PDF 변환/병합

| 모드 | 설명 |
|------|------|
| 단일 변환 | 특정 DOCX 파일 → PDF |
| 일괄 변환 | 폴더 내 전체 DOCX → PDF |
| 변환 후 병합 | 일괄 변환 후 모든 PDF를 하나로 병합 |

---

### 탭 4 — HWP 처리

HWP 파일들을 XLSX DB로 변환합니다.

1. 입력 폴더 설정 (HWP 파일들이 있는 폴더)
2. 출력 XLSX 파일 경로 지정
3. "HWP → XLSX 변환" 클릭

**추출되는 필드**: 파일명, 보관회사명, 작성일자, 관리번호, 분류, 현 보관처, 제작처, MODEL명, 양산처, 품명, 도번번호, 금형규격, 금형재질(BASE/CORE), CAVITY수, 금형수명, GATE형식, 사용기계, 계약일, 승인일, 금형가

---

### 탭 5 — 이미지 추출

1. 입력 폴더 설정 (HWP 파일 폴더)
2. 출력 폴더 설정 (이미지 저장 폴더)
3. "이미지 추출" 클릭

**파일명 규칙**: `品名_図番.jpg` (도번 없으면 `品名.jpg`, 이미지 여러 개면 `品名_2.jpg`)

---

## 신규 이력카드 발행 (★ 버튼)

메인 화면 상단 "★ 신규 이력카드 발행" 버튼을 클릭합니다.

**다이얼로그 구성**

- **직전 항목 표시**: DB 마지막 행의 파일명과 품명 표시 (연번 확인용)
- **File name**: XLSX에서 자동 계산한 다음 연번 (직접 수정 가능)
- **필수 항목**: 품명, 도번번호, 관리번호
- **선택 항목**: 보관회사명, 작성일자, 승인일, 분류, 제작처, MODEL명, 양산처, 금형규격, CAVITY수
- **체크박스**: 新作 / 増作 / 二元化 / 業者更変 / 機種更変
- **금형 사진**: 이미지 파일 첨부 (선택)

"발행" 클릭 시:
1. XLSX DB에 새 행 추가
2. 연번 폴더에 Word 파일 생성 (`출력폴더/XX-NNN.docx`)
3. 사진 첨부 시 `.data/` 폴더에 이미지 복사

---

## 이미지 탐색 순서

Word 생성/재생성 시 아래 순서로 이미지를 탐색합니다:

1. XLSX `金型写真` 컬럼 값
2. `図番番号` 값
3. `品名` 값
4. 연번 (out_name)
5. `.data/` 폴더 (신규 발행 첨부 이미지)

각 단계에서 trailing underscore/hyphen 자동 제거 후 재탐색합니다.

```

```

## 설정 파일 (config.yaml)

경로 설정과 창 위치/크기가 자동 저장됩니다. 프로그램 종료 시 현재 값으로 갱신되므로 별도 수정 불필요합니다.

수동으로 수정하고 싶다면 프로그램 종료 후 텍스트 에디터로 편집하세요.

## 트러블슈팅

**Q: "입력 폴더를 찾을 수 없음"**  
→ "찾기..." 버튼으로 실제 경로를 다시 지정하세요.

**Q: HWP 데이터가 일부 누락됨**  
→ HWP 양식이 표준 금형이력카드 형식인지 확인하세요. 필드 레이아웃이 다르면 파싱 오류가 발생할 수 있습니다.

**Q: Word 생성 후 이미지가 없음**  
→ 이미지 폴더 경로가 올바른지 확인하고, 이미지 파일명이 `品名_図番.jpg` 또는 `品名.jpg` 형식인지 확인하세요.

**Q: 이력 저장 시 "XLSX 저장 실패"**  
→ 해당 XLSX 파일이 Excel 등 다른 프로그램에서 열려 있습니다. 닫고 다시 시도하세요.

**Q: PDF 변환 실패**  
→ Microsoft Word가 설치되어 있어야 합니다. Word를 설치하거나 다른 PC에서 실행하세요.
