# 금형이력카드 처리 프로그램 - 사용 설명서

## 📖 개요

이 프로그램은 한글(HWP) 형식의 금형이력카드에서 데이터와 이미지를 자동으로 추출하여 엑셀 파일로 변환한 후, 템플릿 워드 문서에 데이터를 자동으로 입력하는 통합 솔루션입니다.

### 프로그램 흐름
```
[1단계] HWP 파일 → 엑셀 변환
   ↓
[2단계] 이미지 추출
   ↓
[3단계] 엑셀 데이터 → 워드 문서 생성
```

---

## 🚀 시작하기

### 설치

```bash
# 1. 필수 패키지 설치
pip install -r requirements.txt

# 2. 프로그램 실행
python main.py
```

### 시스템 요구사항
- **Python**: 3.7 이상
- **메모리**: 최소 1GB (권장 2GB+)
- **디스크**: 입력 파일 크기의 3배 정도 여유 공간

## 📝 3단계 사용 방법

### **1단계: HWP 파일 처리** 
`한글파일 처리` 탭

#### 목표
한글 형식의 금형이력카드(HWP) 파일들을 엑셀 파일로 변환합니다.

#### 사용 방법

1. **입력 폴더 설정**
   - "찾기..." 버튼을 클릭하여 HWP 파일들이 있는 폴더 선택
   - 또는 기본 폴더 `YES/` 사용
   - 해당 폴더에 19-001.hwp, 19-002.hwp 등의 파일이 있어야 함

2. **출력 파일 설정**
   - 생성될 엑셀 파일의 경로와 이름 지정
   - 기본값: `data/output/output_from_hwp.xlsx`

3. **"HWP → XLSX 변환" 버튼 클릭**
   - 처리 시간은 파일 개수에 따라 다름 (105개 기준 약 30초)
   - 진행 상황은 하단 "작업 로그"에서 확인 가능

#### 추출되는 데이터 필드
- 파일명, 보관회사명, 작성일자, 관리번호
- 분류, 현 보관처, 제작처, MODEL명
- 양산처, 품명, 도번번호, 금형규격
- 금형재질(BASE, CORE), CAVITY 수
- 금형수명, GATE 형식, 사용기계
- 계약일, 승인일, 금형가

#### ⚠️ 주의사항
- HWP 파일은 반드시 한글로 작성된 양식이어야 함
- 폴더 경로에 특수문자가 없어야 함
- 같은 이름의 엑셀 파일이 있으면 덮어써짐

---

### **2단계: 이미지 추출**
`한글파일에서 이미지 추출` 탭

#### 목표
HWP 파일에 포함된 금형 사진(이미지)을 자동으로 추출합니다.

#### 사용 방법

1. **입력 폴더 설정**
   - 1단계와 동일한 폴더 선택 (HWP 파일이 있는 폴더)
   - 기본값: `YES/`

2. **출력 폴더 설정**
   - 추출된 이미지를 저장할 폴더 지정
   - 기본값: `img/`
   - 폴더가 없으면 자동 생성

3. **"이미지 추출" 버튼 클릭**
   - PNG, JPG 등 다양한 형식 지원
   - 파일명 형식: `19-001.jpg`, `19-001_2.jpg` (여러 이미지인 경우)

#### 추출 결과
- 105개 HWP 파일에서 약 109개 이미지 추출 예상
- 이미지 크기: 일반적으로 100KB~500KB

#### 💡 팁
- 이 단계는 선택사항 (이미지가 필요 없으면 건너뛸 수 있음)
- 이미지가 없는 HWP 파일은 무시됨
- 기존 이미지가 있으면 덮어써짐

---

### **3단계: 워드 문서 생성**
`엑셀 파일로부터 워드파일 생성` 탭

#### 목표
1단계에서 생성한 엑셀 파일의 데이터를 템플릿 워드 문서에 자동으로 입력하여 개별 문서를 생성합니다.

#### 사용 방법

1. **엑셀 파일**
   - 1단계에서 생성한 엑셀 파일 선택
   - 기본값: `data/output/output_from_hwp.xlsx`

2. **템플릿 워드 파일**
   - 데이터를 입력할 기준이 되는 워드 문서 선택
   - 기본값: `data/templates/11-000_양식.docx`
   - 이 파일은 수정되지 않음 (원본 유지)

3. **이미지 폴더** (선택)
   - 2단계에서 추출한 이미지 폴더 지정
   - 기본값: `img/`
   - 이미지 없이 진행 가능

4. **출력 폴더**
   - 생성된 워드 문서들을 저장할 폴더
   - 기본값: `data/output/`

5. **처리 행 제한** (선택)
   - 엑셀의 처음 N개 행만 처리
   - 0 = 전체 처리 (기본값)
   - 테스트 시 5-10 정도로 설정해서 먼저 확인 가능

6. **"DOCX 생성" 버튼 클릭**
   - 처리 시간: 파일당 약 0.5~1초
   - 105개 파일 기준 약 1-2분 소요

#### 생성 결과
- 파일명: `19-001.docx`, `19-002.docx` 등
- 각 파일은 엑셀의 해당 행 데이터로 채워짐
- 이미지가 있으면 자동 삽입됨

#### 템플릿 문서 준비
템플릿 워드 문서에는 다음과 같이 플레이스홀더를 작성해야 합니다:

```
{保管会社名}        → 보관 회사명
{作成日子}          → 작성 날짜
{管理番号}          → 관리 번호
{分 類}             → 분 류
{現 保管処}         → 현 보관처
{製作処}            → 제작처
{MODEL名}           → MODEL명
{量産処}            → 양산처
{品 名}             → 품 명
{図番番号}          → 도번번호
{金型規格}          → 금형규격
{金型材質-BASE}     → 금형재질-BASE
{金型材質-CORE}     → 금형재질-CORE
{CAVITY 数}         → CAVITY 수
{金型寿命}          → 금형수명
{GATE 型式}         → GATE 형식
{使用機械}          → 사용기계
{契約日}            → 계약일
{承認日}            → 승인일
{金型価}            → 금형가
{金型写真}          → 금형 사진 (이미지 자동 삽입)
```

## 🔧 트러블슈팅

### Q: "입력 폴더를 찾을 수 없음" 오류
**원인**: 지정한 폴더가 존재하지 않음
**해결**:
1. "찾기..." 버튼으로 실제 폴더 경로 확인
2. 폴더명에 한글이 포함되어 있는지 확인
3. 폴더에 읽기 권한이 있는지 확인

### Q: HWP 파일이 인식되지 않음
**원인**: 파일이 손상되었거나 다른 형식
**해결**:
1. 파일을 한글로 열어서 정상 여부 확인
2. 파일 확장자가 .hwp인지 확인
3. 폴더의 다른 파일부터 테스트

### Q: 데이터가 빠져 있거나 잘못됨
**원인**: HWP 양식이 표준과 다름 또는 데이터 미입력
**해결**:
1. HWP 문서의 양식이 표준 금형이력카드 형식인지 확인
2. 필수 필드가 모두 입력되어 있는지 확인
3. 특수문자나 줄바꿈이 많은 부분 확인

### Q: 이미지가 추출되지 않음
**원인**: HWP에 이미지가 없거나 손상됨
**해결**:
1. HWP 파일을 열어서 이미지 존재 확인
2. 로그에서 추출된 이미지 개수 확인
3. 이미지가 없는 파일은 무시되므로 정상

### Q: 워드 문서 생성이 느림
**원인**: 파일 개수가 많거나 시스템 성능 부족
**해결**:
1. 먼저 적은 수의 파일로 테스트 (처리 행 제한 사용)
2. 다른 프로그램 종료
3. 메모리 부족 시 시스템 재부팅

## 💡 활용 팁

### 작은 규모로 먼저 테스트
```
1. 처리 행 제한을 5로 설정
2. "DOCX 생성" 실행
3. 결과 확인 후 0으로 변경해서 전체 처리
```

### 이미지 재추출
```
1. img 폴더 삭제 (또는 이름 변경)
2. 2단계 이미지 추출 다시 실행
3. 3단계 워드 문서 재생성
```

### 템플릿 수정
```
1. data/templates/11-000_양식.docx 파일을 워드로 열기
2. 필요한 플레이스홀더 추가/수정
3. 저장 후 3단계 실행
```

### 로그 확인
```
• 작업 로그 창에서 실시간으로 진행 상황 확인
• "[X/Y]" 형식: 현재/전체 진행률
• "✓ 완료" 메시지로 성공 여부 판단
```

## 🔐 데이터 보안

- **원본 파일 안전**: 1단계 및 2단계 실행 후에도 원본 HWP 파일은 수정되지 않음
